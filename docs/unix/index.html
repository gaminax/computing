Unix

The Unix [1] Operating System consists of a kernel, and associated software.

Linux is one of the Kernels.

GNU ( Gnu’s not Unix ) is one implementation of the associated software.

Unix originated from Bell Labs in the 1970s.

The Unix OS has many derivates, among which are Linux, and Mac OS. Apple macOS is a variant of Berkeley BSD Unix.

Unix operating systems are written in C.
